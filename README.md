# Trader Behavior vs Market Sentiment Analysis

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index)
and trader behavior using historical Hyperliquid trading data.

## Objective
To understand how market sentiment impacts trader performance, trade direction,
and profitability, and to extract actionable insights for sentiment-aware trading strategies.

## Datasets
- Hyperliquid historical trader data
- Bitcoin Fear & Greed Index

## Key Insights
- Traders exhibit more aggressive directional behavior during Greed phases
- Fear periods show reduced win rates but relatively controlled losses
- Certain accounts remain consistently profitable across sentiment regimes

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## How to Run
1. Place both CSV files in the project directory
2. Run the notebook top to bottom
